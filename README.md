# Real-Time Chess Game Simulation

A web-based chess game simulation focused on real-time synchronization and multiplayer interaction. The project implements live communication features between players using Pusher and ngrok for real-time event broadcasting and external access.

## Technologies Used
- PHP
- MySQL
- JavaScript
- Pusher
- ngrok
- HTML
- CSS

## Features
- Real-time game synchronization
- Multiplayer interaction system
- Chess board simulation
- Database integration with MySQL
- Live event updates using Pusher
- External local server exposure using ngrok

## Description
This project was developed as a practice application focused on real-time communication and backend integration. The system simulates an online chess environment where players can interact through synchronized game events and live updates.

Although the game is partially functional, the primary objective of the project was to explore real-time technologies, event-driven communication, and multiplayer synchronization concepts.

## Learning Objectives
- Real-time event handling
- Client-server communication
- Backend integration with MySQL
- Multiplayer synchronization concepts
- Dynamic frontend updates
- Networking and external server exposure
